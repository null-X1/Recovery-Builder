# تعليمات تشغيل تطبيق MoodyMe 📱

## 📋 المتطلبات الأساسية

### المتصفحات المدعومة:
- ✅ **Chrome** (الإصدار 80 أو أحدث)
- ✅ **Firefox** (الإصدار 75 أو أحدث)
- ✅ **Safari** (الإصدار 13 أو أحدث)
- ✅ **Edge** (الإصدار 80 أو أحدث)

### المميزات الإضافية:
- 🎤 **تسجيل الصوت**: يتطلب صلاحية الميكروفون
- 📸 **رفع الصور**: يعمل مع جميع المتصفحات
- 📱 **تثبيت PWA**: متاح على Chrome وEdge

## 🚀 طرق التشغيل

### 1. التشغيل المحلي (بدون خادم)
```bash
# افتح المجلد وانقر مرتين على
index.html
```

### 2. التشغيل مع خادم محلي (مُفضل)

#### باستخدام Python:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# ثم افتح: http://localhost:8000
```

#### باستخدام Node.js:
```bash
# تثبيت live-server
npm install -g live-server

# تشغيل الخادم
live-server

# أو باستخدام serve
npx serve
```

#### باستخدام PHP:
```bash
php -S localhost:8000
```

### 3. النشر على الويب

#### GitHub Pages:
1. ارفع الملفات إلى repository على GitHub
2. اذهب إلى Settings > Pages
3. اختر source: Deploy from branch
4. اختر branch: main
5. احفظ وانتظر النشر

#### Netlify:
1. اسحب المجلد إلى موقع netlify.com
2. أو اربط GitHub repository
3. سيتم النشر تلقائياً

#### Vercel:
```bash
# تثبيت Vercel CLI
npm i -g vercel

# نشر التطبيق
vercel
```

## 📱 تثبيت التطبيق كـ PWA

### على Chrome (Android/Desktop):
1. افتح التطبيق في Chrome
2. ابحث عن رمز "تثبيت" في شريط العنوان
3. أو اضغط على القائمة > "تثبيت التطبيق"
4. اتبع التعليمات

### على Safari (iOS):
1. افتح التطبيق في Safari
2. اضغط على زر المشاركة (مربع مع سهم)
3. اختر "إضافة إلى الشاشة الرئيسية"
4. أدخل اسم التطبيق واضغط "إضافة"

### على Edge:
1. افتح التطبيق في Edge
2. اضغط على القائمة (ثلاث نقاط)
3. اختر "التطبيقات" > "تثبيت هذا الموقع كتطبيق"

## 🔧 استكشاف الأخطاء وإصلاحها

### المشاكل الشائعة:

#### 1. الرسوم المتحركة لا تعمل
**الحل:**
- تأكد من الاتصال بالإنترنت
- امسح cache المتصفح
- أعد تحميل الصفحة

#### 2. تسجيل الصوت لا يعمل
**الحل:**
- تأكد من إعطاء صلاحية الميكروفون
- استخدم HTTPS (مطلوب لتسجيل الصوت)
- تحقق من إعدادات المتصفح

#### 3. البيانات لا تُحفظ
**الحل:**
- تأكد من تفعيل LocalStorage في المتصفح
- لا تستخدم الوضع الخاص (Incognito)
- تحقق من مساحة التخزين المتاحة

#### 4. التطبيق لا يظهر بالعربية بشكل صحيح
**الحل:**
- تأكد من دعم المتصفح للخط العربي
- تحقق من اتصال الإنترنت لتحميل خط Cairo
- أعد تحميل الصفحة

### مسح البيانات:
```javascript
// افتح Developer Tools (F12) واكتب:
localStorage.clear();
location.reload();
```

## 🎯 دليل الاستخدام السريع

### الصفحة الرئيسية:
1. **اختر مزاجك** من البطاقات الست
2. **اضغط على البطاقة** للانتقال لصفحة التسجيل

### صفحة التسجيل:
1. **اكتب ملاحظاتك** في المربع النصي
2. **أضف صورة** (اختياري) بالضغط على زر الكاميرا
3. **سجل صوت** (اختياري) بالضغط على زر الميكروفون
4. **احفظ المدخل** بالضغط على "احفظ يومي"

### صفحة التاريخ:
- **تصفح** إدخالاتك السابقة
- **اضغط على الصور** لعرضها بحجم كبير
- **شغل التسجيلات الصوتية** مباشرة

### صفحة الإحصائيات:
- **شاهد الرسم البياني** لأنماط مزاجك
- **راجع النسب المئوية** لكل حالة مزاجية

### صفحة الإعدادات:
- **فعّل الوضع الليلي** للراحة البصرية
- **فعّل قفل التطبيق** لحماية خصوصيتك
- **امسح البيانات** عند الحاجة

## 🔐 الأمان والخصوصية

### حماية البيانات:
- جميع البيانات محفوظة **محلياً** على جهازك
- لا يتم إرسال أي بيانات لخوادم خارجية
- يمكنك حذف جميع البيانات في أي وقت

### نصائح الأمان:
- استخدم قفل التطبيق في الأجهزة المشتركة
- احفظ نسخة احتياطية من بياناتك بانتظام
- لا تشارك جهازك مع الآخرين بدون قفل

## 📊 حدود التخزين

### LocalStorage:
- **الحد الأقصى**: 5-10 ميجابايت لكل موقع
- **تقدير السعة**:
  - نص: ~10,000 إدخال
  - صور: ~50-100 صورة متوسطة الحجم
  - صوت: ~10-20 تسجيل صوتي

### نصائح لتوفير المساحة:
- ضغط الصور قبل الرفع
- احذف الإدخالات القديمة غير المهمة
- استخدم تسجيلات صوتية قصيرة

## 🔄 النسخ الاحتياطي

### تصدير البيانات (يدوياً):
```javascript
// افتح Developer Tools (F12) واكتب:
const data = localStorage.getItem('moodEntries');
const blob = new Blob([data], {type: 'application/json'});
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = 'moodyme-backup.json';
a.click();
```

### استيراد البيانات:
- قريباً سيتم إضافة هذه الميزة في التحديثات القادمة
